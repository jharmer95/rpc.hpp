Detailed Description

template<typename Serial>
class rpc::serial_adapter< Serial >

Class that allows any serialization type to interact with rpc.hpp.

The user can provide their own implementations of this class for a serialization type or use one of the built-in variants (found in the 'apaters' directory)

Template Parameters
SerialContaining class for the serialization library
